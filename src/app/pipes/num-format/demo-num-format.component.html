Number Format

Examples

String

  • {{ 1555.55 | numFormat: '#,##0.0000' }}
  • {{ 1555.55 | numFormat: '#,##0' }}
  • {{ 1555.55 | numFormat: '###0' }}
  • {{ 51555.55 | numFormat: '###,###0.00' }}
  • {{ 51555.555 | numFormat: '###,##0.00' : 'fr' }}
  • {{ 51555.555 | numFormat: '###,##0.00' : 'fr' : 'trunc' }}

Intl.NumberFormatOptions

  • {{ 51555.55 | numFormat: { style: 'currency', currency: 'EUR' } : 'fr-FR' }}
